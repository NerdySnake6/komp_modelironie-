# S3 Object Storage Менеджер для Yandex Cloud

Простой веб-интерфейс на Flask, позволяющий управлять файлами в бакете Yandex Object Storage (совместимом с S3).

## Возможности

- Просмотр списка файлов в бакете
- Загрузка файлов
- Удаление файлов
- Получение временной presign-ссылки для просмотра/скачивания (на 1 час)

## Технологии

- Python 3
- Flask (веб-фреймворк)
- boto3 (AWS SDK для Python, работает с Yandex Object Storage)
- HTML/CSS (без JS — минимализм)
- .env для хранения секретов
